What is HTML?

HTML (Hyper Text Markup Language) is used to lay out the structure of a webpage.

HTML Example

Example Explained

HTML elements are the building blocks of HTML pages.

  • The <!DOCTYPE html> declaration defines this document to be HTML5
  • The <html> element is the root element of an HTML page
  • The lang attribute  defines the language of the document
  • The <head> element is a container for metadata which typically define the title, character set, styles, scripts, and other meta information.
  • The <meta> element contains meta information about the document
  • The charset attribure defines the character set used in the document
  • The <title> element specifies a title for the document
  • The <body> element contains the visible page content
  • The <h1> element defines a large heading
  • The <p> element defines a paragraph

Common HTML tags

  • <head></head>
  • <title></title>
  • <body></body>

  • contents of website
  • metadata about the page that is useful for the browser when displaying the page
  • title of the page
  • body of the page
  • header (h1 is the largest header, h6 is the smallest header)
  • <ul></ul>
  • <ol></ol>
  • <li></li>
  • unordered list
  • ordered list
  • list item (must be inside either <ul></ul> or <ol></ol>)
<img src="..." height="200" width="300">

NOTE: This is a single tag without an end tag.

  • image stored at src attribute, whcih can also be a URL

  • both height and width are optional (if one is omitted, the browser will auto-size the image), and can also take a percentage: height=50% to automatically scale the image to a certain portion of the page
